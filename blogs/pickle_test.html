Store Python Class Objects In Pickle File¶

In many cases we need to store in files such that we do not need a communication between processes. Here we test the module pickle.

Simple list case test¶

In [2]:
import pickle
with open('pickle_test.pkl', 'wb') as file:
    list1 = [{'a':1}, (1,2), [3,4]]
    pickle.dump(list1, file, True)
    file.close()
with open('pickle_test.pkl', 'rb') as file:
    list2 = pickle.load(file)
    print ('The loaded list is:\n', list2)
    file.close()

The loaded list is:
 [{'a': 1}, (1, 2), [3, 4]]

import networkx as nx
import pickle
def pickle_g(nodes, regular):
    graph = nx.random_regular_graph(regular, nodes, seed = 3)
    with open('pickle_graph.pkl', 'wb') as file:
        pickle.dump(graph, file, True)
        file.close()
    with open('pickle_graph.pkl', 'rb') as file:
        load_graph = pickle.load(file)
        file.close()
